# Solaris Energia - Versão CNPJ 44.430.230/0001-66

Esta pasta contém uma versão do site da **Solaris Energia** voltada para condomínios residenciais e comerciais, associada ao CNPJ **44.430.230/0001-66**.

## Visão Geral

O conteúdo foi adaptado para síndicos, administradoras e conselhos de moradores que desejam avaliar a implantação de energia solar em áreas comuns, com foco em redução de custos e valorização patrimonial.

-   **Tecnologias:** HTML5, CSS3, JavaScript (ES6+)
-   **Design:** Responsivo, com identidade visual limpa e sustentável.
-   **Uso típico:** Landing para campanhas focadas em condomínios e apresentações em assembleias.
